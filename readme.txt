Проект базы данных с тестовыми данными для разрабатываемого мобильного приложения Нумизматика.
База для сортировки и колекционирования монет, с указанием множества параметров (Страна, монетный двор, металл, год выпуска, фото аверса, реверса и т.п.)
Приложение с возможностью создания сообществ для общения и новостей.
